#232. 多项式除法

内存限制：256 MiB 时间限制：4000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者： 单敬博
提交 提交记录 统计 测试数据 讨论

题目描述

给出一个次多项式和次多项式

求和，使。其中的次数不超过。

输入格式

第一行两个整数和，为和的项数

第二行个整数为从低次到高次的系数。

第三行个整数位从低次到高次的系数。

输出格式

第一行个整数为从低次到高次的系数

第二行个整数为从低次到高次的系数

样例

样例输入

12 8
2 0 0 1 1 0 2 1 7 1 0 2
2 0 0 2 0 4 2 1

样例输出

998244304 21 1 998244349 2
100 998244311 998244351 107 998244308 194 24
